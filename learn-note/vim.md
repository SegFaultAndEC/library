| 键        | 效果         |
| -------- | ---------- |
| \<C-u>   | 往上翻半页      |
| \<C-d>   | 往下翻半页      |
| \<C-b>   | 往上翻一页      |
| \<C-f>   | 往下翻一页      |
| zz       | 光标到屏幕中间    |
| zt       | 光标到屏幕第一行   |
| zb       | 光标到屏幕最后一行  |
| m{char}  | 标记当前位置     |
| \`{char} | 跳转到标记位置    |
| f{char}  | 跳转当前行的字符   |
| .        | 重复上次操作     |
| ci(      | 修改某个括号内的内容 |
| gu       | 转小写        |
| gU       | 转大写        |
| g~       | 大小写反转      |
