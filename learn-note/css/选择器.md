```html
<!doctype html>

    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">

</head>
<body>

    <p class="classSelector">类选择器</p>
    <p id="idSelector">id选择器</p>
</body>

```
## 元素选择器
直接元素名加花括号
```css
h4{
	color:red;
}
```
## 类选择器
```css
.classSelector{
	color:blue;
}
```
## ID选择器
```css
#idSelector{
	color:green;
}
```
